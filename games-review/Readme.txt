READ ME
XAMPP stands for Cross-Platform (X), Apache (A), MySQL (M), PHP (P) and Perl (P). It is a simple, lightweight Apache distribution that makes it extremely easy for developers to create a local web server for testing purposes.
XAMPP is a free and open-source cross-platform web server solution stack package developed by Apache Friends, consisting mainly of the Apache HTTP Server, MariaDB database, and interpreters for scripts written in the PHP and Perl programming languages.
In Order to connect the Website change your localhost port to 82 from XAMPP.

GUIDE TO CHANGE THE PORT:
Open XAMPP.
Click on Config
Than Click on Apache(httpd.Config)
After that click on the Edit from navbar.
Click on Find.
write 80 then it will find the port 80 for you and then just change the port 80 to 82 and save it.


RUN The Website 
Now enter localhost:82/game-review on google chrome in order to run the Website.
The website is about the games review where you can view the latest games and comments on then.
